# blocks_game

ブロックス（Blokus）を参考にした、2人対戦のブラウザゲームです。

## 遊び方

1. `index.html` をブラウザで開く（またはローカルサーバーで配信する）。
2. 手番プレイヤーがピースを選択し、盤面をクリックして配置する。
3. 同色ピースは **辺で接してはいけない**、**角でのみ接続できる**。
4. 最初の一手は開始角（青: 左上、黄: 右下）を含める必要がある。
5. 置けないときは「パス」。両者連続でパスするとゲーム終了。
6. 置いたマス数が多いプレイヤーの勝ち。

## 操作

- 回転 ↻: 選択中ピースを90度回転
- 反転 ⇋: 選択中ピースを左右反転
- パス: 手番をスキップ
- リスタート: 新しいゲームを開始

## ローカル実行例

```bash
python3 -m http.server 4173
```

その後 `http://localhost:4173` を開いてプレイできます。
